# مجلد صور الموقع

ضع الصور هنا بنفس الأسماء التالية وسيلتقطها الموقع تلقائياً.
إن لم توجد الصورة، يعرض الموقع بديلاً أنيقاً (تدرّج + أيقونة) بدون أي خطأ.

|--------|----------------|---------|
| `hero/slide-1.jpg` | 1920×1080 (أفقية) | الشريحة الأولى |
| `hero/slide-2.jpg` | 1920×1080 | الشريحة الثانية |
